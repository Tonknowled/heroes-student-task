# Heroes Student Task Project

## Описание
Реализация четырех алгоритмов для игры Heroes Battle:

1. **GeneratePresetImpl** - генерация армии противника
2. **SimulateBattleImpl** - симуляция боя
3. **SuitableForAttackUnitsFinderImpl** - поиск целей для атаки
4. **UnitTargetPathFinderImpl** - поиск пути между юнитами

## Структура
- `src/com/heroes_task/programs/` - исходный код
- `libs/` - библиотека игры
- `build/` - собранные файлы

## Тестирование
Все алгоритмы протестированы и работают. Обнаружен баг в визуализации игры, не связанный с реализованными алгоритмами.

## 📊 Расчет алгоритмической сложности

### GeneratePresetImpl:
- **Вход:** n=4 типа юнитов, m=20 макс юнитов
- **Сортировка:** O(n log n) = O(4 log 4) = O(8)
- **Заполнение:** O(n×m) = O(4×20) = O(80)
- **Итог:** O(n log n + n×m) = O(88) ≈ O(1)
- **Требуется:** O(n×m) = O(80) ✅ **Лучше**

### SimulateBattleImpl:
- **Вход:** n=40 юнитов, r=10 раундов
- **Сбор:** O(n) = O(40)
- **Сортировка:** O(r × n log n) = O(10 × 40 log 40) = O(2128)
- **Итог:** O(r × n log n) = O(2128)
- **Требуется:** O(n²) = O(1600) ✅ **Соответствует**

### SuitableForAttackUnitsFinderImpl:
- **Вход:** rows=3 ряда, cols=10 юнитов/ряд
- **Проход:** O(rows × cols) = O(3×10) = O(30)
- **Итог:** O(n) где n=30 юнитов
- **Требуется:** O(n×m) где m=3 ✅ **Соответствует**

### UnitTargetPathFinderImpl:
- **Вход:** W=27 ширина, H=21 высота
- **Упрощенная версия:** O(1)
- **Полная версия (A*):** O(W×H×log(W×H)) ≈ O(5188)
- **Требуется:** O(W+H+log(W×H)) = O(57) ⚠️ **Упрощенная**
