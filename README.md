# Heroes Student Task Project

## Описание
Реализация четырех алгоритмов для игры Heroes Battle:

1. **GeneratePresetImpl** - генерация армии противника
2. **SimulateBattleImpl** - симуляция боя
3. **SuitableForAttackUnitsFinderImpl** - поиск целей для атаки
4. **UnitTargetPathFinderImpl** - поиск пути между юнитами

## Структура
- `src/com/heroes_task/programs/` - исходный код
- `libs/` - библиотека игры
- `build/` - собранные файлы

## Тестирование
Все алгоритмы протестированы и работают. Обнаружен баг в визуализации игры, не связанный с реализованными алгоритмами.
